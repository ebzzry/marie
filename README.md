marie
=====


Overview
--------

This is a collection of Common Lisp utilities.


Usage
-----

Clone this repo to `~/common-lisp`:

    $ mkdir -p ~/common-lisp
    $ git clone https://github.com/ebzzry/marie ~/common-lisp/marie

Then load it:

    * (ql:quickload :marie)

To return the symbols exported by this package:

    * (marie:symbols :marie)
